About Wave Executor

Wave Executor is a script executor (injection tool) for PC used in Roblox to run Lua/LuaU scripts. It enables cheats like auto-farms, ESP, speed, teleport, and supports custom hub scripts and mods for popular games.

Key features
  • Stable injection engine for many Roblox builds
  • Support for community hubs and custom scripts
  • Lightweight, minimal UI, fast startup
