üîç What is a UserScript?

A UserScript is a JavaScript program that modifies web pages to enhance functionality, automate tasks, or customize your browsing experience. Think of it as a mini-extension that runs directly in your browser to improve how websites work for you.

UserScripts are lightweight, fast, and highly customizable solutions for power users who want more control over their web experience.

Important Notice: Google Chrome for Android does not support browser extensions, including userscript managers. This is a deliberate limitation set by Google for mobile browsers.

Android users must install alternative browsers like Firefox, Yandex Browser, Samsung Internet, or Lemur Browser to use UserScripts. These browsers fully support extensions and provide the same functionality as their desktop counterparts.
